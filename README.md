### 💻 Featured Projects

#### 🏗️ [Concrete Crack Detection & Severity Classification](#) *(Thesis Project)*
A hybrid machine learning pipeline designed to evaluate structural integrity by automating crack detection and severity assessment.
- Implemented **YOLOv8** for high-precision real-time crack detection and **U-Net** for detailed crack segmentation.
- Engineered a hybrid CNN–Transformer model architecture to accurately classify crack severity.
- **Stack:** Python · YOLOv8 · U-Net · CNN · Transformers

#### 📱 [FOCUOS – Mobile Usage & Productivity Manager](https://github.com/rjregero03/FocusOS_app)
A mobile application engineered to manage screen time, build healthier digital habits, and calculate productivity scores.
- Implemented a custom rule-based classification algorithm to analyze app usage behavior and generate dynamic productivity scoring.
- Real-time app usage monitoring with a dedicated focus mode, backed by Firebase for dynamic analytics.
- **Stack:** Flutter · Dart · Firebase · Rule-Based Algorithms

#### 🌐 [Centralized Base Management System (YTR)](https://github.com/rjregero03/ytr-system)
A role-based web platform built to automate record management, membership tracking, and event management for a local organization.
- Designed Role-Based Access Control (RBAC) across distinct officer roles with scoped permissions.
- Developed an interactive live dashboard pulling real-time data from MySQL for optimal officer visibility.
- Includes a "Wisdom Archive" module utilizing AI-powered search for document retrieval and summarization.
- **Stack:** Laravel · MySQL · Tailwind CSS · AI Integration

#### 🧮 [Advanced Scientific Calculator](#)
A custom-built calculator application featuring an advanced mathematical parser.
- Engineered custom logic from scratch to accurately process complex operations, including exponents, nested parentheses, and trigonometry.
